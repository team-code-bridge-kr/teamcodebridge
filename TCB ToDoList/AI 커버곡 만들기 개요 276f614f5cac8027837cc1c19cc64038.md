## **미니 코딩(코랩) 예시 — 보컬 분리**

—음 학생들한테 “소리가 그림(스펙트로그램)으로 보인다” 체험용.

```python
# Colab 첫 셀
```

```python
# 간단 분리 & 시각화
from spleeter.separator import Separator
import librosa, librosa.display, soundfile as sf
import matplotlib.pyplot as plt

# 1) 오디오 로드
y, sr = librosa.load('/content/input_song.mp3', sr=16000)

# 2) 파형 미리보기
plt.figure()
librosa.display.waveshow(y, sr=sr)
plt.title('Waveform'); plt.show()

# 3) Spleeter로 보컬/반주 분리(2 stems: voice + accompaniment)
separator = Separator('spleeter:2stems')    # 2/4/5 stems 가능
prediction = separator.separate(y.reshape(1, -1))

vocals = prediction['vocals'].squeeze()
accomp = prediction['accompaniment'].squeeze()

sf.write('vocals.wav', vocals, sr)
sf.write('instrumental.wav', accomp, sr)

# 4) 스펙트로그램 보기(“소리의 색칠공부”)
S = librosa.amplitude_to_db(librosa.stft(y), ref=librosa.amplitude_to_db)
plt.figure()
librosa.display.specshow(S, sr=sr, x_axis='time', y_axis='log')
plt.title('Spectrogram'); plt.colorbar(); plt.show()
```

## **이론 설명(중학생 교육과정 참고함)**

- **소리=파동**: 빠르게 진동하는 공기. 컴퓨터는 소리를 **숫자**로 저장(샘플링).
- **스펙트럼/스펙트로그램**: 프리즘처럼 소리를 **주파수별 색깔**로 펼친 그림.
- **딥러닝이 목소리를 흉내내는 법**: 많은 예시(목소리)를 듣고, 그 사람만의 **음색/억양 패턴**을 숫자로 배움. 새 노래가 들어오면 그 **패턴대로** 다시 말/노래를 만들어 줌. —음 성대모사 연습생 비유로 이해시키기. (정보과목)
- **주의**: 유명인의 목소리/원곡은 **권리 이슈**가 있어. 수업은 **학습 목적**으로만, 외부 업로드/상업 사용 금지. (기술과 과정 관련)
