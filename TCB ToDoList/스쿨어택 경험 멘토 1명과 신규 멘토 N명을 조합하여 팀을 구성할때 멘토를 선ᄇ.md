# 스쿨어택 경험 멘토 1명과 신규 멘토 N명을 조합하여 팀을 구성할때 멘토를 선발·배치하는 기준 및 방안

# 기본 멘토 선발 순서

1. 운영지원팀 : 학교와 컨텍하여 일정 조율 및 참여 학생 수 확정
2. 학생 수 및 학년을 고려하여 몇개의 팀으로 나누어 멘토링을 진행할 것인지 확정
3. 날짜 및 장소를 공유하여 스쿨어택 참여 희망 멘토 수요조사
4. 1개 팀에 최소 경험 멘토 1명 필수 + 신규 멘토 N명 (경험멘토 참여 우선권 부여?? )  

# 멘토 팀 구성 방안

1. 멘토링 사전에 참여 학생들을 대상으로 희망 프로젝트 진행 분야 수요조사

  (ex) 웹페이지 개발, 사물인터넷, 프로그램 개발, 빅데이터 등등)

1. 수요조사 결과를 바탕으로 관련 전공 멘토 우선선발하여 배치

# 신규멘토 사전교육 방안

참여 확정 멘토들은 스쿨어택 전 1회 대면 만남 → 멘토링 계획 수립 + 신규멘토 교육

교육목차

  1. 기본적인 멘토링 진행 순서

1. 프로젝트 진행 중 노션 활용 방안
2. 피드백 방법
3. 발표 피피티 구성 내용

등등 많을 것 같은데 이러한 것들을 기존 사례를 활용해서 교육문서를 하나 따로 만들어서 스쿨어택 점점 하면서 수정해나가면서 신규멘토님들 교육에 활요하면 좋을 듯 합니다!!

# 경험멘토 및 신규멘토 역할분배 방안

경험멘토 : 학생들과의 의사소통, 타 팀 팀장들과의 각 팀의 상황 공유, 신규 멘토 지원, 피드백 제공 등등

신규맨토 : 피드백 제공, 프로그램 운영 지원 등등
