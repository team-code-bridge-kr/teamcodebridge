1. **핵심 개념 정리**
    1. **바이브 코딩(ViB Coding)**
        - 자연어 프롬프트로 AI가 코드 생성
        - 코딩 진입 장벽 완화, 문제 해결·논리 설계 집중

        - 명확·구체·맥락을 갖춘 지시문 작성
        - 결과 형태(코드·리포트 등) 명시

        - ChatGPT API 연동
            
            ```python
            # 1) OpenAI 라이브러리 설치 (터미널에서 한 번만)
            # pip install openai
            
            import os
            import openai
            
            # 2) 환경변수에 API 키 설정
            #    Windows: set OPENAI_API_KEY="발급받은_키"
            #    Mac/Linux: export OPENAI_API_KEY="발급받은_키"
            openai.api_key = os.getenv("OPENAI_API_KEY")
            
            def ask_gpt(prompt: str, model="gpt-4"):
                """프롬프트를 보내고 응답 텍스트를 반환하는 함수"""
                resp = openai.ChatCompletion.create(
                    model=model,
                    messages=[{"role":"user", "content": prompt}],
                    temperature=0.7,
                    max_tokens=300
                )
                return resp.choices[0].message.content
            
            # 3) 간단한 예시 호출
            if __name__ == "__main__":
                sample = "진자의 길이가 1.0m일 때 주기를 계산하는 수식을 알려줘."
                answer = ask_gpt(sample)
                print("GPT 응답:\n", answer)
            ```

3. **과목별 실험 개요**
    1. **물리: 단진자 주기 측정**
        - 가설: “진자 길이↑ → 주기↑”
        - 데이터 수집: 3가지 길이별 T 측정
        - 분석: g = 4π²L/T² 계산 및 그래프
        - GPT 활용: 가설 조언, 코드 생성, 결과 해석
            
            ```python
            lengths = [0.5, 1.0, 1.5]                 # 진자 길이 (m)
            periods = [1.42, 2.01, 2.47]              # 1회 주기 T (s)
            
            import math
            # 각 데이터에서 g 계산: g = 4π²L / T²
            g_values = [(4 * math.pi**2 * L) / (T**2) for L, T in zip(lengths, periods)]
            print("각 길이별 추정 g 값:", g_values)
            
            # 그래프로 확인하기
            import matplotlib.pyplot as plt
            plt.plot(lengths, g_values, 'o-')
            plt.xlabel('진자 길이 (m)')
            plt.ylabel('추정된 g (m/s²)')
            plt.title('길이에 따른 중력가속도 g 추정값')
            plt.show()
            ```
            
    2. **화학: 온도별 반응 속도**
        - 가설: “온도↑ → 반응 속도↑”
        - 실험: 발포정제 용해 시간 측정(10℃·25℃·40℃·60℃)
        - 분석: 시간 역수 그래프
        - GPT 활용: 실험 설계, 코드 생성, 이론 설명
            
            ```python
            temps = [10, 25, 40, 60]          # 온도(℃)
            times = [90, 60, 35, 20]          # 반응 완료 시간(초)
            rates = [1/t for t in times]      # 반응 속도 = 시간의 역수
            
            import matplotlib.pyplot as plt
            plt.plot(temps, rates, 'o-')
            plt.xlabel('온도 (℃)')
            plt.ylabel('반응 속도 (1/초)')
            plt.show()
            ```
            
    3. **생명과학: 광합성 속도 vs 빛 세기**
        - 가설: “빛 세기↑ → 광합성↑”
        - 실험: 수초 기포 개수 측정(거리 10cm·20cm·30cm·40cm)
        - 분석: 거리-기포 그래프, 포화점 토의
        - GPT 활용: 절차 제안, 코드 생성, 개념 해설
            
            ```python
            distances = [10, 20, 30, 40]      # 램프-수초 거리(cm)
            bubbles   = [40, 15, 8, 5]        # 1분당 기포 수
            
            import matplotlib.pyplot as plt
            plt.plot(distances, bubbles, 'o--')
            plt.xlabel('거리 (cm)')
            plt.ylabel('기포 수 (개/분)')
            plt.show()
            ```
            
    4. **지구과학: 충돌 크레이터 크기**
        - 가설: “높이↑ → 크레이터 지름↑”
        - 실험: 밀가루판에 구슬 낙하(50cm·100cm·150cm)
        - 분석: 산점도 및 회귀 분석
        - GPT 활용: 실험 방법, 코드 생성, 에너지 이론 설명
            
            ```python
            heights   = [50, 100, 150]        # 낙하 높이(cm)
            diameters = [4.2, 5.8, 7.1]       # 크레이터 지름(cm)
            
            import numpy as np
            import matplotlib.pyplot as plt
            
            coef = np.polyfit(heights, diameters, 1)
            plt.scatter(heights, diameters)
            plt.plot(heights, np.poly1d(coef)(heights), '--')
            plt.xlabel('높이 (cm)')
            plt.ylabel('지름 (cm)')
            plt.show()
            ```
